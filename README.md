### Setup New Machine

```
chmod +x ~/dotfiles/osx/setup-a-new-machine.sh
sh ~/dotfiles/osx/setup-a-new-machine.sh

```

# Local Bin

```sh
chmod -R +x ~/dotfiles/macos/setup-custom-bin.sh
sh ~/dotfiles/macos/setup-custom-bin.sh
```
